Example

const injector: Injector =
    ReflectiveInjector.resolveAndCreate([{provide: 'validToken', useValue: 'Value'}]);
expect(injector.get('validToken')).toEqual('Value');
expect(() => injector.get('invalidToken')).toThrowError();
expect(injector.get('invalidToken', 'notFound')).toEqual('notFound');

Injector returns itself when given Injector as a token:

const injector = ReflectiveInjector.resolveAndCreate([]);
expect(injector.get(Injector)).toBe(injector);

Class Details

get(token: Type<T>|InjectionToken<T>, notFoundValue?: T) : T

Retrieves an instance from the injector based on the provided token. If not found:

  • Throws NoProviderError if no notFoundValue that is not equal to Injector.THROW_IF_NOT_FOUND is given
  • Returns the notFoundValue otherwise
