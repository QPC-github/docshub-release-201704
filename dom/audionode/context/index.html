AudioNode.context

The read-only context property of the AudioNode interface returns the associated AudioContext, that is the object representing the processing graph the node is participating in.

Syntax

var aContext = anAudioNode.context;

Value

An AudioContext object.

Example

var AudioContext = window.AudioContext || window.webkitAudioContext;

var audioCtx = new AudioContext();

var oscillator = audioCtx.createOscillator();
var gainNode = audioCtx.createGain();

oscillator.connect(gainNode);
gainNode.connect(audioCtx.destination);

oscillator.context;
