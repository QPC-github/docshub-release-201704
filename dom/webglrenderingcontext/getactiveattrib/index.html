WebGLRenderingContext.getActiveAttrib

The WebGLRenderingContext.getActiveAttrib() method of the WebGL API returns a WebGLActiveInfo object containing size, type, and name of a vertex attribute at a given position in a WebGLProgram.

Syntax

WebGLActiveInfo gl.getActiveAttrib(program,index);

Parameters

program
A WebGLProgram containing the vertex attribute.
index
A GLuint specifying the index of the vertex attribute to get.

Return value

A WebGLActiveInfo object.

Examples

gl.getActiveAttrib(program, i);
