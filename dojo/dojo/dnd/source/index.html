dojo/dnd/Source

dojo/dnd/Selector

Summary

a Source object, which can be used as a DnD source, or a DnD target

Usage

var foo = new Source(node,params);

dojo/dnd/Source

Parameter Type Description
node DOMNode | String

node or node's id to build the source on
params Object
Optional

any property of this class may be configured via the params object which is mixed-in to the dojo/dnd/Source instance

See the dojo/dnd/Source reference documentation for more information.
