clear()source public 

clear( boolean $check )

Delete all keys from the cache. This will clear every cache config using APC.

Parameters

boolean $check

If true, nothing will be cleared, as entries are removed from APC as they expired. This flag is really only used by FileEngine.

Returns

boolean
