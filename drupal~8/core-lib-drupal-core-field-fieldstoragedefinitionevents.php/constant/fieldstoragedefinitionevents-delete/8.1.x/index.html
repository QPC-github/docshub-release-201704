constant FieldStorageDefinitionEvents::DELETE

Name of the event triggered for field storage definition deletion.

This event allows you to respond anytime a field storage definition is deleted. The event listener method receives a \Drupal\Core\Field\FieldStorageDefinitionEvent instance.
