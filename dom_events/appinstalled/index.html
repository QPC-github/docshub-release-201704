appinstalled

The appinstalled event of the Web Manifest API is fired when the browser has successfully installed a page as an application.

Bubbles No
Cancelable No
Target objects Window
Interface Event

Examples

function handleInstall(ev) {
  console.log('Thank you for installing our app!'); 
} 

window.onappinstalled = handleInstall;

Inheritance

The appinstalled event implements the Event interface — it has available the properties and methods defined on this interface.
