GeneratorFunction.prototype

The GeneratorFunction.prototype property represents the GeneratorFunction prototype object.

Description

GeneratorFunction objects inherit from GeneratorFunction.prototype. GeneratorFunction.prototype cannot be modified.

Properties

GeneratorFunction.constructor
The initial value is GeneratorFunction.
GeneratorFunction.prototype.prototype
The value is %GeneratorPrototype%.
