function overlay_get_regions_to_render

overlay_get_regions_to_render()

Returns a list of page regions that rendering should be limited to.

Return value

An array containing the names of the regions that will be rendered when drupal_render_page() is called. If empty, then no limits will be imposed, and all regions of the page will be rendered.
