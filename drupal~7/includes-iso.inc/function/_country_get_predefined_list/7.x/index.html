function _country_get_predefined_list

_country_get_predefined_list()

Get an array of all country code => country name pairs.

Get an array of all country code => country name pairs as laid out in ISO 3166-1 alpha-2. Grabbed from location project (http://drupal.org/project/location).

Return value

An array of all country code => country name pairs.

File

includes/iso.inc, line 17
Provides a list of countries and languages based on ISO standards.
