AfterViewInit

Stable Class

What it does

Lifecycle hook that is called after a component's view has been fully initialized.

How to use

.

@Component({selector: 'my-cmp', template: `...`})
class MyComponent implements AfterViewInit {
  ngAfterViewInit() {
    // ...
  }
}
