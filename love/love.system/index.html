love.system

Available since LÖVE 0.9.0
This module is not supported in earlier versions.

Provides access to information about the user's system.

Functions

love.system.getClipboardText Gets text from the clipboard. 0.9.0
love.system.getOS Gets the current operating system. 0.9.0
love.system.getPowerInfo Gets information about the system's power supply. 0.9.0
love.system.getProcessorCount Gets the number of CPU cores in the system. 0.9.0
love.system.openURL Opens a URL with the user's web or file browser. 0.9.1
love.system.setClipboardText Puts text in the clipboard. 0.9.0
love.system.vibrate Causes the device to vibrate, if possible. 0.10.0
