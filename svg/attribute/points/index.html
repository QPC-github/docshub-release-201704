points

The points attribute defines a list of points required to draw a <polyline> or <polygon> element.

Each point is defined by a X and a Y coordinate in the user coordinate system. It's a common practice (but not required) to use a comma to mark out the X and Y coordinate of each point and to use space to mark out each point.

Usage context

Categories None
Value <list-of-points>
Animatable Yes
Normative document SVG 1.1 (2nd Edition): The polyline element
SVG 1.1 (2nd Edition): The polygon element

Example

<svg xmlns="http://www.w3.org/2000/svg" version="1.1">
  <polygon points="100,10 250,150 200,110"
           style="fill:lime;stroke:purple;stroke-width:1" />
</svg>

Elements

The following elements can use the points attribute

  • <polyline>
  • <polygon>
