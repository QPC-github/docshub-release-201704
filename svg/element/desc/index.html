<desc>

Each container element or graphics element in an SVG drawing can supply a description string using the <desc> element where the description is text-only.

When the current SVG document fragment is rendered as SVG on visual media, <desc> elements are not rendered as part of the graphics. Alternate presentations are possible, both visual and aural, which display the <desc> element but do not display <path> elements or other graphics elements. The <desc> element generally improves accessibility of SVG documents.
