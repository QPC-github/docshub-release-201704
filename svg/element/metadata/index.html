<metadata>

The <metadata> SVG element allows to add metadata to SVG content. Metadata is structured information about data. The contents of <metadata> elements should be elements from other XML namespaces such as RDF, FOAF, etc.

Usage context

Categories Descriptive element
Permitted content Any elements or character data

Attributes

Global attributes

  • Core attributes
  • Global event attributes

Specific attributes

None

DOM Interface

This element implements the SVGMetadataElement interface.
