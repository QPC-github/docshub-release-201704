public __construct(DOMElement $node, string $currentUri, string $method = null, string $baseHref = null)

Constructor.

Parameters

DOMElement $node A \DOMElement instance
string $currentUri The URI of the page where the form is embedded
string $method The method to use for the link (if null, it defaults to the method defined by the form)
string $baseHref The URI of the used for relative links, but not for empty action

Exceptions

LogicException if the node is not a button inside a form tag

public string getMethod()

Gets the form method.

If no method is defined in the form, GET is returned.

Return Value

string The method

public string getUri()

Gets the URI of the form.

The returned URI is not the same as the form "action" attribute. This method merges the value if the method is GET to mimics browser behavior.

Return Value

string The URI
